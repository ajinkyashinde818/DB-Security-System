# DBSecuritySystem

A secure, Java-based desktop application built using **Java Swing** and **JDBC** that manages and protects access to a database system. It supports user login authentication, encrypted credentials, access logging, and admin controls to monitor database interactions.

---

## 🔐 Features

- 🔑 Secure User Login System (Username + Password)
- 🔒 Encrypted Database Credential Storage
- 📊 Admin Dashboard with Access Logs
- 🧾 Download Activity Logs (with JFileChooser)
- 🕵️ Detects DB Login Events from MySQL Workbench Logs
- ⚙️ Supports Database Operations with Role-Based Access
- 📁 Backup and Restore Database Files

---

## 🛠️ Technologies Used

- **Java (Swing GUI)**
- **JDBC (Database Connectivity)**
- **MySQL Database**
- **Java File I/O** (for reading MySQL logs)
- **JFileChooser** for file selection and download
- **NetBeans IDE** (recommended)
